# flowercrown-blog

Multi-tenant Subscription System「flowercrown」の開発記ブログ。
Hugo + PaperMod、GitHub Pages(Actions デプロイ)。

- 公開 URL: https://ko-chan.github.io/flowercrown-blog/
- 前身: [saru-blog](https://github.com/ko-chan/saru-blog)(アーカイブ済み。Pages は配信継続)

## ローカルプレビュー

```sh
git submodule update --init
hugo server -D
```

## 運用ルール

記事は「実際に歩けるマイルストーンを出荷した後に 1 本ずつ」。
コンテンツカレンダー・連載ロードマップは作らない(前身ブログの教訓)。
