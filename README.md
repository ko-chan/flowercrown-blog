# flowercrown-blog

Multi-tenant Subscription System「flowercrown」の開発記ブログ。
Hugo + PaperMod、GitHub Pages(Actions デプロイ)。

- 公開 URL: https://ko-chan.github.io/flowercrown-blog/
- 前身: [saru-blog](https://github.com/ko-chan/saru-blog)(アーカイブ済み。Pages は配信継続)

## ローカルプレビュー

```sh
git submodule update --init
hugo server -D
```

## 運用ルール

記事は不定期。「実際に歩けるマイルストーンを出荷した後に 1 本ずつ」で、
総回数・コンテンツカレンダー・連載ロードマップは定めない(前身ブログの教訓)。

転載は Zenn のみ(公開のたびに手動で 1 回)。dev.to / Hashnode / はてな / Bluesky へは
出さない — 前身ブログの実測で Zenn 以外は閲覧数が伸びなかったため。自動化もしない。
